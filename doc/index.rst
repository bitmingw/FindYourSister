Welcome to FindYourSister's documentation!
==========================================

This is the documentation of FindYourSister. This project is a simple
object detection framework for computer vision.
It utilizes 2D features and point mapping method for detection.
It provides a set of interfaces to quickly setup your own algorithms
and programs based on the framework.

This project is constructed under the supervision of
Prof. Ma Hongbin at Beijing Institute of Technology. If you are
unclear about the functions or if you find some bugs, please tell us at
https://github.com/bitmingw/FindYourSister/issues.
Pull requests are also welcome!
